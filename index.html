In this project, we're going to build some expanding cards. We would style these panels to have a background image and a heading. We will use Flexbox to align everything and then add a bit of JavaScript so that when we click one of these, it changes the class to active and then expands with a CSS transition. So we'll be using JavaScript for the event. Listeners will be using CSS transitions to give it a smooth, thorough effect. You will not see the heading coming until the card is expanded fully.
